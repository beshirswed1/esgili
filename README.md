## 📖 About The Project

**Eşgili Restoran** is an ultra-modern, fully responsive web application designed to elevate the dining experience. Built for a prominent restaurant in Gaziantep, this project features a real-time digital menu, an advanced shopping cart, and a highly secure, luxurious Admin Dashboard for live management. 

It is engineered with **Advanced SEO (JSON-LD, Open Graph)** to ensure top-tier visibility on Google and social media platforms.

## ✨ Key Features

* **⚡ Real-Time Synchronization:** Menu updates, price changes, and availability are synced instantly using **Firebase Firestore**.
* **🛒 Smart Cart System:** Seamless state management using **Redux Toolkit** for a flawless ordering experience.
* **👑 Legendary Admin Dashboard:** A Glassmorphism-inspired, secure, and visually stunning command center to manage categories, products, and landing page content.
* **📱 Ultra Responsive UI:** Pixel-perfect design that looks incredible on mobile phones, tablets, and desktop screens.
* **🔍 SEO Monster:** Fully optimized with structured data (Schema.org), dynamic metadata, and Open Graph configurations.
* **🎨 Stunning Visuals:** Fluid animations, hover effects, and premium iconography using **Lucide React** & **FontAwesome**.

## 🛠 Tech Stack

This project is built using the best-in-class technologies:

### Core Frameworks & Languages
* **[Next.js (App Router)](https://nextjs.org/)** - React framework for SSR and optimal performance.
* **[React 18](https://react.dev/)** - UI Library.
* **[TypeScript](https://www.typescriptlang.org/)** & **JavaScript** - For robust, type-safe code.

### State Management & Database
* **[Redux Toolkit](https://redux-toolkit.js.org/)** - Global state management (Cart, Restaurant Info).
* **[Firebase](https://firebase.google.com/)** - Authentication & Real-time NoSQL Database.

### Styling & UI
* **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS framework for rapid UI development.
* **[Lucide React](https://lucide.dev/)** & **[FontAwesome](https://fontawesome.com/)** - Premium SVG icons.
* **Next/Font** - Optimized loading for *Inter* and *Playfair Display* fonts.
